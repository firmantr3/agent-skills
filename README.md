# Agent Skills

A collection of reusable **skills** for [Kilo](https://kilo.ai) and compatible AI coding assistants. Each skill injects domain-specific instructions and workflows to make agents smarter at particular tasks.

## Skills

| Skill | Description |
|-------|-------------|
| [**clarify-first**](clarify-first/) | Prompt ambiguity gate — stops agents from guessing when a request is vague, asks clarifying questions before acting. |
| [**critical-advisor**](critical-advisor/) | Adversarial thinking partner that challenges ideas, plans, and designs before helping refine them. |
| [**plan-kiro**](plan-kiro/) | Spec-driven development: turns a plain-language feature description into structured specs under `.kiro/specs/`. |
| [**execute-kiro**](execute-kiro/) | Executes `tasks.md` generated by plan-kiro — environment discovery, constraint loading, spec-driven implementation with fallback plans. |
| [**step-guide**](step-guide/) | Interactive step-by-step guide for hands-on technical tasks — installs, cloud setup, server config, CLI workflows. |
| [**goal-loop**](goal-loop/) | Autonomous execution loop that iterates until a defined goal is fully satisfied. |
| [**code-orchestrator**](code-orchestrator/) | Routes user tasks to the right subagent to minimize API calls. |

## Usage

Skills are loaded by the agent on demand. Most trigger automatically when the task matches their description (e.g., "review this plan" triggers critical-advisor). Some require explicit invocation.

## Install via skillfish

Install all skills in one command — works with Claude Code, Cursor, Copilot, Windsurf, and 28+ other agents:

```bash
# Install everything (detects all supported agents automatically)
npx skillfish add firmantr3/agent-skills --all

# Or pick specific skills interactively
npx skillfish add firmantr3/agent-skills

# Single skill by name
npx skillfish add firmantr3/agent-skills clarify-first
npx skillfish add firmantr3/agent-skills goal-loop
npx skillfish add firmantr3/agent-skills plan-kiro

# Add skillfish.json to your repo for team sync
# skillfish bundle && git add skillfish.json
```
